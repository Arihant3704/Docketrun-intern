The Data Lab: Dataset & Annotation

Before a model can learn, it needs data. This data comes from sources like Kaggle, HuggingFace, or GitHub, often in formats like .png or .jpg.

1. Dataset Acquisition

Data is collected from videos and images to train models.

Kaggle HuggingFace GitHub COCO Dataset

2. Annotation Process

Labeling is crucial. It involves defining "Ground Truth" for the computer. For object classification, this often means drawing a bounding box.

Training & Pre-trained Models

Training is the process where the model learns by fine-tuning its parameters over many "epochs" (cycles). We often start with pre-trained models to save time.

Concept: Fine Tuning

We use high epochs and large datasets to adjust the model's internal weights.

Epochs 100

Pre-trained Models

Instead of starting from scratch, we use models already trained on massive datasets.

  • .pt → YOLO (Object Detection)
  • .caffe → MobileNet SSD

Live Metric
