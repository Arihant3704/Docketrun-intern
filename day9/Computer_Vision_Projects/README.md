# Computer Vision Projects

This directory contains essential Computer Vision projects that are fundamental for a Computer Engineer's portfolio. Each project focuses on a core concept in the field.

## Projects List

### 1. [Face Detection](./01_Face_Detection)
**Concept**: Object Detection (Faces)
**Tools**: OpenCV (Haar Cascades or DNN)
**Description**: Real-time face detection system. Good starting point for understanding cascades and pre-trained models.

### 2. [Object Detection (YOLO)](./02_Object_Detection_YOLO)
**Concept**: Deep Learning based Object Detection
**Tools**: YOLO (You Only Look Once), PyTorch/Ultralytics
**Description**: State-of-the-art object detection capable of identifying multiple object classes in images or video streams.

### 3. [Image Segmentation](./03_Image_Segmentation)
**Concept**: Semantic/Instance Segmentation
**Tools**: Unet, Mask R-CNN, or similar
**Description**: Partitioning an image into segments to locate objects and boundaries (pixels belonging to a specific class).

### 4. [OCR Scanner](./04_OCR_Scanner)
**Concept**: Optical Character Recognition
**Tools**: Tesseract, OpenCV
**Description**: Extracting text from images. Useful for document digitization and license plate recognition.

### 5. [Hand Gesture Recognition](./05_Hand_Gesture_Recognition)
**Concept**: Pose Estimation / Tracking
**Tools**: MediaPipe
**Description**: Tracking hand landmarks to recognize gestures (e.g., counting fingers, controlling volume).

### 6. [Lane Detection](./06_Lane_Detection)
**Concept**: Feature Extraction, Computer Vision for Autonomous Driving
**Tools**: OpenCV (Canny Edge, Hough Transform)
**Description**: Identifying lane markings on a road, a critical component of autonomous vehicle systems.

### 7. [AI Virtual Painter](./07_AI_Virtual_Painter)
**Concept**: Human-Computer Interaction (HCI)
**Tools**: MediaPipe, OpenCV, NumPy
**Description**: "Air Canvas" application allowing users to draw on the screen using hand gestures. Features selection mode (2 fingers) and drawing mode (1 finger).

### 8. [Virtual Try-On](./08_Virtual_Try_On)
**Concept**: Augmented Reality (AR)
**Tools**: MediaPipe Pose, OpenCV
**Description**: Real-time AR system that overlays virtual clothing (e.g., shirts) onto the user's body by tracking skeletal landmarks.
